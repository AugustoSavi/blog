Sou engenheiro de software focado no ecossistema **Java** e na arquitetura de sistemas distribuídos de alta escala. Atualmente, atuo como **Senior Java Software Engineer na FCamara**, onde trabalho no desenvolvimento de sistemas bancários críticos, focando em gestão transacional e investimentos.

Minha abordagem de engenharia é pautada pela busca por escalabilidade, observabilidade e qualidade de código, sempre aplicando padrões como **DDD**, **Clean Architecture** e **API First**.

## Trajetória Profissional

Minha jornada na tecnologia é marcada por uma evolução contínua:

- **FCamara (Atual):** Atuação em canais bancários, evoluindo microsserviços com Java (11 a 25), Spring Boot e Apache Camel, além de garantir a saúde operacional através de observabilidade profunda.

- **Betha Sistemas:** Durante 5 anos, percorri desde o assistente de desenvolvimento até o cargo de **Arquiteto de Sistema**. Fui responsável por definir padrões arquiteturais *cloud-native*, estruturar pipelines de CI/CD e liderar a modernização de sistemas legados para microsserviços.
- **Acadêmico:** Sou graduado em **Ciência da Computação** pela UNESC (Universidade do Extremo Sul Catarinense).

## Stack Técnica Principal

- **Linguagens:** Java (8–25), JavaScript e Python
- **Frameworks:** Spring Boot, Micronaut, Angular e React.
- **Cloud & Infra:** AWS, GCP, Docker, Kubernetes e Terraform
- **Mensageria:** SNS, SQS, RabbitMQ, Kafka.
- **Bancos de Dados:** PostgreSQL, MySQL e Mongo. 
- **Observabilidade:** Datadog, Prometheus, Grafana.
